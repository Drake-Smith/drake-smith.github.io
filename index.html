Chef's On Me

Full Stack MERN App matches Private Chef's with Users

Chef's On Me

View Live Site
View GitHub

I participated in Startup Weekend - Los Angeles, an event that gathers entrepreneurs, developers, marketers, and designers together into groups to create a company over the course of one weekend. The company I was a part of was Chef's On Me, a platform that allows users to conveniently search for and hire private chefs at an affordable price.

Because of the extremely limited amount of time, my goal, as part of the development team, was to create a Minimum Viable Product that could be easily scaled for the future. Over the course of 1.5 days, myself and two other developers built a working version of Chef's On Me with the MERN stack (Mongo, Express, React, Node). The structure of the company's web application was in place so that, with more time, we can continue adding features to the site. We also used React so that we could later create an iOS and Android app with React Native.

Built with:
  • MongoDB
  • React
  • Firebase
  • HTML
  • JSX
  • Express
  • Node
  • CSS
  • Bootstrap
